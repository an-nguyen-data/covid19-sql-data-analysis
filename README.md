# Covid-19-Data-Analysis (SQL Data Exploration Project)

## Project objective
This project aims to analyze global Covid-19 data using SQL to uncover trends in infections, deaths, and vaccinations. The goal is to gain insights into how the pandemic spread across countries and how vaccination efforts impacted populations.

## Project Insight
- The highest infection rates were identified by comparing total cases to population, showing which countries were most affected relative to their size.
- Total death counts by country highlight which regions experienced the most severe outcomes during the pandemic.
- The calculated death percentage reveals differences in mortality across countries.
- Rolling vaccination data using window functions shows how vaccination numbers accumulated over time in each country.
- Combining deaths and vaccination data through joins provides a clearer view of how vaccination progress relates to pandemic trends.

## Final conclusion:
The analysis shows that Covid-19 impacted countries differently depending on population size and healthcare conditions. Countries with higher infection rates relative to their population experienced more severe spread, while death percentages varied across regions. Vaccination efforts increased steadily over time, with some countries achieving faster rollout. Overall, combining infection, death, and vaccination data provides a clearer understanding of how the pandemic evolved and highlights the importance of vaccination in controlling its impact.
